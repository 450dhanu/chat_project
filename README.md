# MyKotlinApp

This is a simple Android Kotlin project.

---

## How to Run the Project

1. **Open Android Studio**  
   If you don't have Android Studio, download and install it from [https://developer.android.com/studio](https://developer.android.com/studio).

2. **Import the Project**  
   - Open Android Studio.  
   - Go to **File > Open** and select the `MyKotlinApp` folder where this project is located.  
   - Wait for Gradle Sync to complete.

3. **Set Up an Emulator or Device**  
   - Open **AVD Manager** from Tools.  
   - Create and start a new Android Virtual Device (Emulator) or connect a physical Android device.

4. **Run the Project**  
   - Select the **app** module in the Run Configuration dropdown (top toolbar).  
   - Click the **Run** button (▶️) or press **Shift + F10**.  
   - Choose your emulator or connected device to launch the app.

5. **See the App Running**  
   - The app will launch, displaying a simple screen with the text **"Hello, Kotlin!"**
